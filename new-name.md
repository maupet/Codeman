# Codeman Rename — Name Exploration

The project has deviated significantly from the original Codeman design and deserves its own identity. Looking for something unique with Icelandic/Nordic mythology influence.

## Leading Candidates: Fáfnir + Smiðja (Forge)

Fáfnir is the dragon from Norse mythology — originally a dwarf transformed into a dragon guarding a hoard of treasure. Reframed for this project: a powerful guardian watching over a hoard of sessions and agents.

All compound names are **completely available** on both npm and GitHub.

| Name | ASCII | Meaning | Notes |
|------|-------|---------|-------|
| **Fáfnissmiðja** | fafnissmidja | "Fáfnir's forge" | Proper Icelandic genitive (Fáfnis + smiðja). Grammatically correct with double-s. |
| **Fáfnismíðja** | fafnismidja | "Fáfnir's smithy" | Simplified, drops one s. Easier to type, still reads correctly to Icelandic speakers. |
| **Fáfnisverki** | fafnisverki | "Fáfnir's works" | More industrial feel. |
| **Fáfnisverksmiðja** | fafnisverksmidja | "Fáfnir's factory" | Full Icelandic compound. Impressive but long. |
| **Fafnir Forge** | fafnirforge | English hybrid | More approachable internationally. |
| **Fafnir Works** | fafnirworks | English hybrid | Clean, professional. |

Note: `fafnir` alone is taken on npm (v1.0.1, 2014) and GitHub (152+ repos).
